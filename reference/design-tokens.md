# Design Tokens - Reference (ГигаЧат Бизнес)

Извлечено из скриншота-образца. Все значения — приближения на основе визуального анализа.

## Цветовая палитра

### Фоны
| Token | Значение | Где используется |
|---|---|---|
| `bg-primary` | `#FFFFFF` | Сайдбар, основная область, правая панель |
| `bg-page` | `#FAFAFA` / `#F7F8FA` | Фон страницы за контентом (если есть) |
| `bg-input` | `#F5F5F5` | Поле ввода сообщений |
| `bg-hover` | `#F3F4F6` (~gray-100) | Hover на навигационных элементах |
| `bg-active` | `#F0F0F0` (~gray-100/200) | Активный элемент в сайдбаре (подсветка) |
| `bg-tag` | `#FFFFFF` | Фон пилюль/тегов навыков |
| `bg-table-header` | `#F9FAFB` (~gray-50) | Заголовок таблицы |

### Текст
| Token | Значение | Где используется |
|---|---|---|
| `text-primary` | `#111827` (~gray-900) | Заголовки, основной текст навигации |
| `text-secondary` | `#6B7280` (~gray-500) | Вторичный текст, подписи секций |
| `text-muted` | `#9CA3AF` (~gray-400) | Плейсхолдеры, неактивные элементы |
| `text-link` | `#111827` | Ссылки в контенте (чёрные, не синие) |

### Акценты
| Token | Значение | Где используется |
|---|---|---|
| `accent-green` | `#22C55E` (~green-500) | Зелёные точки статуса (навыки, шаги) |
| `accent-green-light` | `#DCFCE7` (~green-100) | Не используется явно, но возможен для hover |

### Границы
| Token | Значение | Где используется |
|---|---|---|
| `border-default` | `#E5E7EB` (~gray-200) | Разделители секций, границы карточек |
| `border-light` | `#F3F4F6` (~gray-100) | Тонкие разделители |
| `border-input` | `#D1D5DB` (~gray-300) | Границы полей ввода |
| `border-tag` | `#E5E7EB` (~gray-200) | Границы пилюль навыков |

## Типографика

### Шрифт
- **Семейство**: Системный sans-serif (визуально похож на Inter / SF Pro / Helvetica Neue)
- **Сглаживание**: antialiased

### Размеры и веса
| Элемент | Размер | Вес | Цвет |
|---|---|---|---|
| Логотип "ГИГАЧАТ БИЗНЕС" | ~14px | 800 (Extra Bold) | `#111827` |
| Навигация — активный | ~14px | 500 (Medium) | `#111827` |
| Навигация — обычный | ~14px | 400 (Regular) | `#111827` |
| Заголовок секции (Детали) | ~16px | 600 (Semibold) | `#111827` |
| Подзаголовок секции (Прогресс, Документы) | ~13px | 500 (Medium) | `#6B7280` |
| Текст контента | ~14px | 400 (Regular) | `#374151` |
| Метка тега навыка | ~13px | 500 (Medium) | `#374151` |
| Мелкий текст / caption | ~12px | 400 (Regular) | `#9CA3AF` |

## Spacing (отступы)

| Элемент | Значение |
|---|---|
| Padding сайдбара | `16px` по горизонтали, `12-16px` по вертикали |
| Gap между nav-элементами | `2-4px` |
| Padding nav-элемента | `8-10px` по вертикали, `12px` по горизонтали |
| Padding правой панели | `16-20px` |
| Padding секции контента | `20-24px` |
| Gap между секциями | `16-20px` |

## Border Radius

| Элемент | Значение |
|---|---|
| Навигационные кнопки | `8px` (rounded-lg) |
| Пилюли навыков | `9999px` (rounded-full) |
| Поле ввода | `12-16px` (rounded-xl / 2xl) |
| Карточки/секции | `12px` (rounded-xl) |
| Аватар пользователя | `9999px` (rounded-full) |

## Shadows

| Элемент | Значение |
|---|---|
| Общий стиль | Минимальные тени, почти flat-дизайн |
| Карточки | Нет видимых теней или очень лёгкий `shadow-sm` |
| Правая панель | Возможно лёгкая тень или только border-left |

## Иконки

- **Стиль**: Линейные (outline), не заливка
- **Размер**: ~20px для навигации, ~16px для вспомогательных
- **Цвет**: Наследуется от текста (`currentColor`)
- **Библиотека**: Визуально похоже на Lucide / Heroicons outline
- **НЕ используются emoji** — только SVG-иконки
