# Diff: Текущий стиль vs. Референс (ГигаЧат Бизнес)

## Ключевые отличия

### 1. Тема сайдбара: ТЁМНАЯ → СВЕТЛАЯ
| Аспект | Текущий | Референс |
|---|---|---|
| Фон сайдбара | `bg-gray-900` (тёмный) | `bg-white` (белый) |
| Текст навигации | `text-gray-400` / `text-white` | `text-gray-700` / `text-gray-900` |
| Разделители | `border-gray-800` | `border-gray-200` |
| Hover | `hover:bg-gray-800/50` | `hover:bg-gray-50` |
| Active | `bg-gray-800 text-white` | `bg-gray-100 text-gray-900` |
| Профиль текст | `text-white` / `text-gray-500` | `text-gray-900` |

### 2. Иконки: EMOJI → SVG
| Текущий | Референс |
|---|---|
| `🏢 📋 📜 🤖 📦 🔧 ⚙️ 👥` | SVG outline иконки (Lucide-style) |
| Встроенные emoji | Монохромные, `currentColor` |

### 3. Акцентный цвет: ОРАНЖЕВЫЙ/ИНДИГО → ЗЕЛЁНЫЙ
| Текущий | Референс |
|---|---|
| `bg-orange-500` (badges, focus rings) | Не используется |
| `bg-indigo-600/20` (gigabrain) | Не используется |
| Нет зелёного акцента | `green-500` — точки статуса, активные навыки |

### 4. Логотип
| Текущий | Референс |
|---|---|
| Градиент-иконка + "Wukong" + подпись | Текстовый логотип "ГИГАЧАТ БИЗНЕС" uppercase bold |

### 5. Структура навигации
| Текущий | Референс |
|---|---|
| Группы: Main modes, Daily, Config, Workspaces, Teams, Settings | Группы: New chat button, Agents, Recent chats, Bottom utils |
| Цветные точки workspaces | Нет цветных маркеров |
| Числовые бейджи (оранжевые/индиго) | Нет числовых бейджей в nav |

### 6. Правая панель
| Текущий | Референс |
|---|---|
| Нет отдельной правой панели "Детали" | Есть панель "Детали" с секциями: Прогресс, Документы, Команды, Навыки |

### 7. Общий визуальный стиль
| Аспект | Текущий | Референс |
|---|---|---|
| Контрастность | Высокая (тёмный sidebar + светлый контент) | Низкая (всё светлое, мягкие акценты) |
| Тени | `shadow-sm`, `shadow-lg` на некоторых элементах | Почти без теней, flat-дизайн |
| Цветовое разнообразие | Orange, Indigo, Blue, Green, Purple, Red | Почти монохром + зелёный акцент |
| "Шумность" | Много цветных элементов, badges, градиенты | Чистый, минималистичный, сдержанный |

## Что НЕ меняется (канва и лейауты)
- Flex layout с sidebar + main content
- Общая структура роутинга/навигации
- Компонентная архитектура React
- Tailwind CSS как инструмент стилизации
- Шрифт Inter
- Основные размеры (sidebar ~240-256px)
